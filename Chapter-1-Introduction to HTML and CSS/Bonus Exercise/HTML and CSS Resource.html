<h1> - <h6>: Headings

Used to define headings. <h1> is the most important, <h6> the least.

<h1>Main Heading</h1>
<h2>Subheading</h2>

<p>: Paragraph

Defines a block of text.

<p>This is a paragraph of text.</p>

<a>: Anchor (Link)

Creates hyperlinks to other pages or sites.

<a href="https://example.com">Visit Example</a>

<img>: Image

Displays an image using the src and alt attributes.

<img src="image.jpg" alt="Description" />

<ul>, <ol>, <li>: Lists

Used to create unordered or ordered lists.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<div> & <span>

<div> is a block-level container. <span> is inline.

<div class="box">Block Content</div>
<span style="color:red;">Inline Text</span>

<header>, <footer>, <main>, <section>, <article>

Semantic tags for better HTML5 structure.

<header>Page Header</header>
<main>
  <section>
    <article>Content</article>
  </section>
</main>
<footer>Page Footer</footer>
