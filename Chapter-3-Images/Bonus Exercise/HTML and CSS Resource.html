<h1> to <h6> - Headings

Used to define HTML headings, from largest (<h1>) to smallest (<h6>).

<h1>Main Heading</h1>
<h2>Subheading</h2>

<p> - Paragraph

Defines a paragraph of text.

<p>This is a paragraph.</p>

<a> - Anchor

Creates a hyperlink to another page or site.

<a href="https://example.com">Visit Example</a>

<img> - Image

Embeds an image in the page.

<img src="image.jpg" alt="Description" width="200" />

<ul>, <ol>, <li> - Lists

Defines unordered and ordered lists.

<ul>
  <li>Item One</li>
  <li>Item Two</li>
</ul>

<div> - Division

A generic container for grouping content.

<div class="container">Content here</div>

<span> - Inline Container

Used to style parts of text inline.

<span class="highlight">Important</span>

<form> - Form

Defines a form for user input.

<form action="submit.php" method="post">
  <input type="text" name="name" />
</form>

<table> - Table

Used to display tabular data.

<table>
  <tr><th>Name</th><th>Age</th></tr>
  <tr><td>Alice</td><td>30</td></tr>
</table>

<code> and <pre> - Code Blocks

Display code snippets with formatting.

<pre><code>Your code here</code></pre>

ðŸŽ¨ CSS Properties

  • color: Sets the text color.
  • background-color: Sets the background color.
  • font-family: Sets the font style.
  • font-size: Adjusts text size.
  • padding: Adds space inside elements.
  • margin: Adds space outside elements.
  • border: Adds borders to elements.
  • text-align: Aligns text (left, right, center).

Example CSS:

body {
  background-color: #f9f9f9;
  font-family: Arial, sans-serif;
  color: #333;
}

h1 {
  font-size: 2rem;
  text-align: center;
  margin-bottom: 1rem;
}
